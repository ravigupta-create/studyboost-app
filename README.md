## What you're looking at

This repository (`studyboost-app`) hosts the **deployed static build** of StudyBoost via GitHub Pages. The TypeScript source code lives in a separate, private repository.

If you've found this repo and want to verify authorship, license, or report a copy of the site somewhere else: open an issue here or on the source repo at <https://github.com/ravigupta-create/studyboost> (private).

## Privacy

StudyBoost runs entirely in your browser. No accounts, no analytics SDKs, no tracking cookies, no telemetry. See [/privacy](https://ravigupta-create.github.io/studyboost-app/privacy) on the live site for full details.
